# Volatrik
Volatrik is a memory forensic framework and perform automated extract digital artifatcs from volatile memory (RAM).

# Installation
How to install & using volatrik :
```
git clone https://github.com/muchtarbas/Volatrik.git
cd Volatrik
chmod +x volatrik.sh
./volatrik.sh
```
